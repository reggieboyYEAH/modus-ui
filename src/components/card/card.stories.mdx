# Card

The `<Card/>` Component.

### Props

<Props of={Card} />

## Default

By default a `<Card/>` is a `block` element so it will take up the whole block

It's up to the children how the content will look, modus-ui will provide a blank slate for flexibility.

<Story name="Default">
  <Card id={'default-card'}>
    <p
      style={{
        display: 'flex',
        margin: '100px',
        justifyContent: 'center',
        fontWeight: 800
      }}
    >
      Faster, Faster, Until the thrill of speed overcomes the fear of death.
      -Hunter S. Thompson
    </p>
  </Card>
</Story>

Source code

```
  <Card id={'default-card'}>
    <p style={{ display: 'flex', margin: '50px', justifyContent: 'center' }}>
      Faster, Faster, Until the thrill of speed overcomes the fear of death. -
      Hunter S. Thompson
    </p>
  </Card>
```
